Modals & Flows

The four critical decision points in the product — export, share, destructive confirm, and upload. Each opens a real, keyboard-accessible dialog built only from system tokens and primitives. Press Esc to dismiss.

Implementation notes
Focus trapped

Tab cycles within the open dialog; focus returns to the trigger on close.

Scrim + emphasized

Backdrop fades at 200ms; dialog scales in on the emphasized curve. Reduced-motion safe.

Token-driven

Zero hardcoded color or spacing — every value resolves through the system contract.
